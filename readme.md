PRORIENTATION FINAL TASK adalah sebuah website tugas akhir yang
menampilkan kumpulan karya dari kelompok PRORIENTATION. Setiap halaman
dirancang untuk membantu pengguna memahami tujuan proyek, melihat hasil
karya tiap kelompok beserta detailnya.

Fitur Website

1.  Halaman Home

-   Menampilkan judul utama sebagai identitas proyek

-   Menggunakan tampilan visual yang menarik

-   Menyediakan navigasi ke seluruh halaman dalam website

2.  Halaman About

-   Menjelaskan tujuan, latar belakang, dan konteks pembuatan website

-   Berisi informasi tentang proyek

-   Tombol panah membantu pengguna scroll otomatis ke bawah atau ke
    > atas.

3.  Halaman Final Task List

-   Menampilkan berbagai pilihan hasil final task kelompok

-   Disajikan dalam bentuk pilihan icon yang mudah dipahami user yang
    > ketika dipencet akan pop-up

4.  Detail Final Task

-   Penjelasan detail tentang karya

-   Icon Sinopsis : menampilkan penjelasan singkat tentang isi atau
    > cerita dari karya tersebut.

-   Icon Credit : menunjukkan daftar anggota atau pihak yang terlibat
    > dalam pembuatan karya.

-   Icon Play MV/Trailer : mengarahkan pengguna ke link video hasil
    > karya.

-   Icon Instagram : membawa pengguna ke akun Instagram resmi terkait
    > karya untuk informasi tambahan.

5.  Halaman Team

-   Menampilkan karakter avatar dari setiap anggota yang jika diklik,
    > dapat diarahkan ke akun Instagram tiap anggota
